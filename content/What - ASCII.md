全称: American Standard Code for Information Interchange
计算机只认识二进制的数据, 二进制的数据由多个二进制位 (bit) 构成, 每个二进制位只有 0 或 1 两种状态, 8 个二进制位就能排列出成 256 种组合, 每种组合表示一个字节 (byte), 基于这个原理, 美国定了一个规则, 规定那种组合对应英语中的哪个ß字符, 这样用户就可以很方便地用自然语言转换成计算机认识的二进制数据, 然后计算机再转换二进制数据为其他用户也能读懂的自然语言, 实现跨设备的信息交换, 这个规则就是 ASCII 编码
